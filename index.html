Implementando um Servidor Web Usando Docker

Passo 1: Crie um repositório público no GitHub (asa-web)

  • git clone https://github.com/seu-usuario/asa-web.git

Passo 2: Crie um Dockerfile

  • # Use uma runtime oficial do nginx como a imagem pai
    FROM nginx:latest
    # Defina o diretório de trabalho no container como /app
    WORKDIR /app
    # Copie o conteúdo do diretório atual para o container em /app
    COPY . /app
    # Substitua o site padrão do nginx
    RUN rm -rf /usr/share/nginx/html/*
    # Copie os arquivos estáticos do seu site para o novo diretório
    COPY . /usr/share/nginx/html
    # Exponha a porta 80 para o host do Docker, para que possamos acessar o servidor web a partir do host
    EXPOSE 80

Passo 4: Crie um script shell para criação do container

  • # Construa a imagem do Docker
    docker build -t asa-web .
    # Execute o container do Docker
    docker run -d -p 8080:80 asa-web
